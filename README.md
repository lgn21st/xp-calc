# XP 兑换中心 (XP Reward Center)

一个简洁、优雅的单文件渐进式 Web 应用 (PWA)，用于在 **Duolingo XP**、**零用钱 (SGD)** 和 **游戏时间** 之间进行快速换算。

## 项目简介

本项目旨在帮助家长和孩子直观地管理奖励制度。通过预设的换算率，输入任意一项数值，应用都会实时自动计算出对应的其他两项数值。

## 核心功能

*   **实时换算**：修改 XP、金额或时间中的任意一项，其他两项会立即同步更新。
*   **移动端优化**：
    *   针对 iOS 设计风格，支持深色模式。
    *   使用 `inputmode="decimal"` 自动调起数字键盘。
    *   支持“添加到主屏幕”，享受全屏 App 体验。
*   **智能格式化**：
    *   自动处理小数点，整数不显示无用的 `.0`。
    *   游戏时间额外提供“小时 + 分钟”的直观读数。
*   **离线可用**：单 HTML 文件设计，包含所有样式和逻辑，无外部依赖，可离线运行。

## 换算逻辑

应用基于**分钟**作为基础单位进行计算：

*   **150 XP = 10 分钟** (即 1 分钟 = 15 XP)
*   **10 分钟 = 3.33 SGD** (即 1 SGD = 3 分钟)

## 技术栈

*   **HTML5**: 结构与 PWA 元标签。
*   **CSS3**: 采用 iOS 设计语言（San Francisco 字体、圆角卡片、毛玻璃效果、系统级深色模式）。
*   **JavaScript**: 原生 JS 实现双向绑定计算、DOM 操作及 Canvas 动态图标生成。

## 如何使用

### 开发者
1.  **修改汇率**：在 `index.html` 的 `<script>` 部分修改 `XP_PER_MIN` 和 `MIN_PER_SGD` 常量。
2.  **部署**：由于是单文件，可以直接将 `index.html` 丢进任何静态托管服务（如 GitHub Pages、Vercel），或直接在浏览器中打开。

### 用户 (移动端推荐)
1.  在 iPhone 上使用 Safari 打开应用链接。
2.  点击分享按钮 -> **“添加到主屏幕”**。
3.  从主屏幕图标启动，即可获得完整的原生应用体验。
